# Module 182: Projected residual model neutrality and collision defects

## 1. Precise theorem / claim being advanced

This module refines the major-arc projected model from Module 181. It separates
projected residual model neutrality into two parts:

```text
generic W-tail neutrality
+ large-prime collision-defect control.
```

The structural claim is that the projected residual model
`Omega_w^{proj}` can be written as a generic collision-free contribution, which
vanishes as `w -> infinity`, plus a collision-defect contribution supported on
congruence collisions among the eight projected vertices. Therefore the
major-arc model-neutrality problem reduces to bounding the averaged
large-prime collision defects.

This module does not prove the required collision-defect bound.
Module 192 later gives a conditional composition lemma for that bound, but
only under explicit structural, overflow, kernel/range, and W-limit
hypotheses.
Module 193 then combines that conditional collision-defect bound with the
generic W-tail calculation to give conditional projected model neutrality.

## 2. Status label

`STRUCTURAL / EXTRACTION`

The generic/collision decomposition is structural. The averaged collision
estimate remains an open analytic input.

## 3. Definitions and variables

Keep the objects from Modules 179-181:

```text
f = nu - 1
B_d(n) = f(n+d) conj(f(n))
c_d = E_n B_d(n)
B_d^circ(n) = B_d(n) - c_d
```

Let the four projected `B_d` slots use kernel shifts:

```text
t = (t0,t1,t2,t3).
```

The eight projected vertices are:

```text
v_{00,0} = n - t0
v_{00,1} = n - t0 + d
v_{10,0} = n + h - t1
v_{10,1} = n + h - t1 + d
v_{01,0} = n + k - t2
v_{01,1} = n + k - t2 + d
v_{11,0} = n + h + k - t3
v_{11,1} = n + h + k - t3 + d.
```

Let `V` be this set of eight affine forms. For `S subset V`, let:

```text
Theta_{w,S}^{proj}(d,h,k;t)
```

be the exact local singular model for the forms in `S`. The projected
residual model is:

```text
Omega_w^{proj}(d,h,k;t)
  = sum_{S subset V} (-1)^(8-|S|) Theta_{w,S}^{proj}(d,h,k;t).
```

For a prime `p>w`, let `pi_p(V)` be the collision partition of `V` modulo
`p`: two vertices are in the same block if their affine forms are congruent
modulo `p`.

Let `Theta_{w,S}^{gen}` denote the same singular model with every projected
vertex treated as distinct modulo every `p>w`. Define the generic residual
model:

```text
Omega_w^{gen}
  = sum_{S subset V} (-1)^(8-|S|) Theta_{w,S}^{gen}.
```

Define the collision defect:

```text
Delta_w^{coll}(d,h,k;t)
  = Omega_w^{proj}(d,h,k;t) - Omega_w^{gen}.
```

## 4. Assumptions

This extraction assumes:

- the W-tricked model removes primes `p <= w`;
- the projected model from Module 181 is the correct major-arc local model;
- exact local factors are used for every subset `S subset V`;
- the projection kernel shifts `t0,t1,t2,t3` are included before local-model
  evaluation;
- collision hyperplanes are described by pairwise differences of vertices in
  `V`.

Analytic closure would require:

- an averaged bound for `Delta_w^{coll}`;
- boundary control for broad projection kernels;
- uniformity in `D`, `R`, `eta`, `w`, and all dyadic ranges.

## 5. Proof / disproof / reduction

Start from the projected residual model:

```text
Omega_w^{proj}
  = sum_{S subset V} (-1)^(8-|S|) Theta_{w,S}^{proj}.
```

Add and subtract the generic model:

```text
Omega_w^{proj}
  = Omega_w^{gen} + Delta_w^{coll}.
```

The generic model is neutral in the W-tail. Indeed, when every vertex is
distinct modulo every `p>w`, each local factor depends only on `|S|` and has
the form:

```text
theta_p(S) = (1 - 1/p)^(-|S|) (1 - |S|/p)
           = 1 + O_{|S|}(1/p^2).
```

Hence, uniformly for `0 <= |S| <= 8`,

```text
Theta_{w,S}^{gen} = 1 + O(sum_{p>w} 1/p^2) = 1 + o_w(1).
```

Therefore:

```text
Omega_w^{gen}
  = sum_{S subset V} (-1)^(8-|S|) (1 + o_w(1))
  = o_w(1),
```

because:

```text
sum_{S subset V} (-1)^(8-|S|) = (1-1)^8 = 0.
```

It remains to control the collision defect. A collision at a prime `p>w`
occurs only if some nonzero pair difference among the eight projected vertices
is divisible by `p`. The relevant pair differences are generated by:

```text
d,
h + t0 - t1,
k + t0 - t2,
h + k + t0 - t3,
h - k + t2 - t1,
k + t1 - t3,
h + t2 - t3,
```

and the corresponding expressions with an added or subtracted `d` where the
two vertices use different `d` endpoints.

Thus model neutrality reduces to proving an averaged collision-defect bound:

```text
(1/D) sum_{D<|d|<=2D}
  E_{h,k,t} |Delta_w^{coll}(d,h,k;t)| = o(1),
```

with the same projection kernel weights as Module 181.

If this bound holds, then:

```text
(1/D) sum_{D<|d|<=2D}
  E_{h,k,t} Omega_w^{proj}(d,h,k;t)
  = o_w(1) + o(1),
```

and the projected residual model is neutral in the required averaged sense,
subject to the usual order of limits.

## 6. Edge cases

- Pair differences equal to zero are structural collisions, not rare
  congruence accidents.
- Kernel shifts can create new collisions even when the unprojected cube is
  collision-free.
- Prime divisors `p>w` of small pair differences are not automatically
  negligible; they must be averaged or excluded.
- A generic W-tail calculation is not enough without collision-defect control.
- The collision-defect average must use the same kernel weights as the major
  projection.
- W-limit order matters: `N -> infinity` first, then `w -> infinity`.
- Boundary transfer can alter the available range of `t` and therefore the
  collision count.
- Smoothed projection kernels need absolute-summability or a separate tail
  estimate before the collision average is meaningful.
